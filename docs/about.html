About

Our Mission & Vision

All We Need is a curated directory of high-quality developer tools, resources, and libraries. In an era of AI-generated content and SEO-spam, finding genuine, engineer-approved tools can be difficult.

We believe in:

  • Quality over Quantity: Only the best tools make it to the list.
  • Community Driven: Anyone can submit a PR to add a tool.
  • No Ads, No Tracking: Just a clean, fast directory.

How it Works

The entire site is open source on GitHub. To contribute:

  1. Fork the repository.
  2. Add a markdown file for your tool in `projects/`.
  3. Submit a Pull Request.

Once merged, you'll arguably appear on our Leaderboard!
