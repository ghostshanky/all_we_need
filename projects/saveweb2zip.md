## Overview
**SaveWeb2Zip** is a powerful online tool that allows you to download and save complete websites or specific landing pages. It packages all necessary assets into a single ZIP file, making it easy to archive or analyze web pages offline.

## Key Features
- **Complete Asset Download**: Captures HTML, CSS, JavaScript, Images, and Fonts.
- **Structure Preservation**: Maintains the original directory structure of the website.
- **Mobile Version**: Option to specifically copy the mobile version of a site.
- **Asset Renaming**: Automatically handles asset renaming for consistent local paths.
- **Simplified Download**: One-click process to archive landing pages or full sites.
- **Free to Use**: Download any page absolutely for free.

## How it Works
1.  Enter the URL of the webpage you want to save.
2.  Select options (Rename assets, Copy mobile version, etc.).
3.  Click **Save** to generate the archive.
4.  Download the ZIP file containing all site files.
